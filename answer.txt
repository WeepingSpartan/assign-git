What is a gitignore file and what should it include for a Java project?

A git ignore file is a text file that tells git which files to ignore in a project. For a java project, by default it includes the compiled class file, log file, BlueJ files, Mobile tools for Java (J2ME), package files and a virtual machine crash log file.
